如果C程序中`a`与`b`是两个有符号整数类型的全局变量，那么下面三种`swap_ab`函数都可以实现交换它们数值的效果。

```c
int a;
int b;

void swap_ab()
{
  int t;
  t = a;
  a = b;
  b = t;
}

void swap_ab_alter1()
{
  swap(&a, &b);
}

void swap_ab_alter2()
{
  swap(&b, &a);
}
```

要描述这样的程序功能，可以使用全局变量的地址与分离逻辑谓词`store`一起构建`Require`条件和`Ensure`条件。

```c
void swap_ab()
/*@ With x y
    Require store_int(&a, x) * store_int(&b, y)
    Ensure store_int(&a, y) * store_int(&b, x) */
;
```
