Longest Increasing Subsequence

Given an integer array nums, return the length of the longest strictly
increasing subsequence.

This verification case uses an explicit dp workspace array supplied by the
caller:

  int lengthOfLIS(int *nums, int numsSize, int *dp)

The C implementation is the standard O(n^2) dynamic programming algorithm:
dp[i] starts at 1, is strengthened from earlier j < i with nums[j] < nums[i],
and the function returns the maximum dp[i].
