Longest Nondecreasing Subsequence

Given an integer array nums, return the length of the longest nondecreasing
subsequence.

This verification case uses an explicit workspace array supplied by the caller:

  int lengthOfLNDS(int *nums, int numsSize, int *tails)

The C implementation is the standard O(n log n) patience sorting algorithm.
The array tails[0..len) is nondecreasing and tails[k] stores a minimal
possible tail value for a nondecreasing subsequence of length k + 1 in the
processed prefix.  For each input value x, binary search finds the first
position whose tail is greater than x; x is appended if no such position
exists, otherwise it replaces that position.
